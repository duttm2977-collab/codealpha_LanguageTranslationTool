# Language Translation Tool

## Description
This project is a simple Language Translation Tool developed using Python and Google Colab. It allows users to translate text into different languages such as Hindi, French, and Spanish.

## Features
- Translate text into multiple languages
- Easy and beginner-friendly
- Uses Google Translator API
- Supports Hindi, French, and Spanish

## Technologies Used
- Python
- Google Colab
- deep-translator
- GitHub

## Sample Languages
- Hindi (hi)
- French (fr)
- Spanish (es)
